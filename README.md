Dev Portfolio Generator is a feature-rich React-based web application designed to help developers quickly create and customize their own personal portfolio websites without needing backend services or design expertise. The application provides an intuitive split-screen interface where users can input their details through a structured form while simultaneously viewing a real-time preview of their portfolio.

The generated portfolio includes all essential sections such as Hero, About, Skills, Projects, Experience, Certifications, Education, Contact, and Footer. Each section is modular, allowing users to dynamically add, remove, or hide sections based on their requirements. This flexibility makes the application suitable for beginners as well as experienced developers who want a clean and customizable portfolio.

One of the key highlights of this project is its advanced customization system. Users can control a global theme (including primary color, background color, and text color) that applies across the entire portfolio. Additionally, each section has optional customization controls, enabling users to override the global theme with their own background colors, text colors, or design styles. Background template selection is also supported, allowing users to choose from predefined design templates for a more visually appealing layout.

The application also includes support for adding certifications with detailed information and optional file uploads (such as images or PDFs). All sections, including certifications and experience, are optional and can be completely removed if not needed, ensuring a clean and relevant final output.

Another standout feature is the export functionality. Users can download their portfolio as a standalone HTML file that preserves all content, styles, selected templates, and layout configurations. The exported file is fully functional offline and can be directly deployed as a personal website.

The project is built using React with a component-based architecture, ensuring maintainability and scalability. It emphasizes clean UI/UX design, efficient state management, and real-world product-like functionality, making it both a practical tool and a strong addition to a developer’s portfolio.
